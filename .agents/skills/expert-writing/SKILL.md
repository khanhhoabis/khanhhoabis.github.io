# Skill: Expert Writing

## Description
Kỹ năng viết lách chuyên gia, kết hợp giữa tư duy phân tích và khả năng truyền đạt.

## Capabilities
- Soạn thảo bài viết dài (Long-form content) chuẩn SEO.
- Sử dụng mô hình Mermaid để vẽ sơ đồ tư duy, quy trình.
- Phân tích và đưa dữ liệu vào bài viết một cách thuyết phục.

## Usage
Sử dụng bởi **Knowledge Engineer** để sản xuất nội dung bài viết.
