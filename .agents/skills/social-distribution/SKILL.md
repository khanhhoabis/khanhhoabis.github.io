# Skill: Social Distribution

## Description
Kỹ năng phân phối nội dung và tối ưu hóa tương tác trên mạng xã hội chuyên gia (LinkedIn).

## Capabilities
- Chuyển đổi bài viết dài thành các bài đăng ngắn (Hooks, Summary).
- Tối ưu hóa SEO Metadata.
- Quản lý phản hồi và thu thập feedback từ cộng đồng.

## Usage
Sử dụng bởi **Growth Catalyst** để phân phối nội dung và tương tác.
