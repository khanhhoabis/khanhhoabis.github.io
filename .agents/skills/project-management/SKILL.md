# Skill: Project Management

## Description
Kỹ năng quản lý công việc thông qua GitHub Projects và Issues.

## Capabilities
- Khởi tạo và quản lý Issue (Title, Description, Labels, Assignees).
- Cập nhật trạng thái trên bảng Kanban của GitHub Project.
- Quản lý Milestones để theo dõi tiến độ các chiến dịch lớn.

## Usage
Sử dụng bởi **Visionary Strategist** để điều phối toàn bộ "tòa soạn".
