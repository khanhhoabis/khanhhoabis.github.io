# Skill: Web Crafting

## Description
Kỹ năng thiết kế giao diện (UI/UX) và lập trình web hiện đại.

## Capabilities
- Thiết kế giao diện bằng HTML/CSS (Vanilla) với phong cách cao cấp.
- Lập trình tính năng tương tác bằng Javascript.
- Tối ưu hóa hiệu suất web và trải nghiệm người dùng trên thiết bị di động.
- Tích hợp các thành phần dữ liệu (biểu đồ, bảng) vào trang web.

## Usage
Sử dụng bởi **Experience Architect** để xây dựng và bảo trì blog.
